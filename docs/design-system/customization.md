# Customization & Migration

Bukie is designed for easy extension and migration.

## Customization
- Extend tokens by updating the CSS variable theme definitions in `src/app/globals.css`
- Add new primitives or override styles with Tailwind utilities, small CSS modules, or shared component abstractions
- Use Storybook to preview and test custom themes and components

## Migration
- Follow usage guidelines for replacing legacy styles and components
- Document migration steps in the design system docs
- Keep accessibility, SSR, and testing requirements in mind
