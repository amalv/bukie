# Vite

Vite is a next-generation front-end build tool that significantly improves the development experience. It's chosen for this project due to its fast hot module replacement (HMR), out-of-the-box support for TypeScript, and its efficient build system.

Vite provides a development server with HMR that updates the browser as soon as you save a file. This makes the feedback loop during development incredibly fast.

Vite's build system uses Rollup for bundling, which is highly efficient and supports tree-shaking to eliminate unused code. This results in smaller and faster production builds.

Vite also has first-class support for TypeScript, meaning you can use TypeScript for both your source code and configuration files. This enhances the development experience by providing type safety and autocompletion.

Moreover, Vite supports CSS, JSON, and other static assets out of the box, and has a rich plugin system for extending its capabilities.
