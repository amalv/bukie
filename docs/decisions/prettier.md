# Prettier

Prettier is an opinionated code formatter that enforces a consistent code style across the entire codebase. 

It helps to maintain code quality by automatically formatting code to adhere to a specific style guide. This eliminates discussions about code style in code reviews and ensures a consistent code style across the project.

Moreover, Prettier supports a wide range of languages and file types, including JavaScript, TypeScript, HTML, CSS, and Markdown, making it suitable for projects with diverse tech stacks.
