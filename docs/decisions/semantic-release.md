# Semantic Release

Semantic Release is an automated version management and package publishing tool. It's chosen for this project due to its ability to automate the whole package release workflow including: determining the next version number, generating the release notes and publishing the package.

Semantic Release uses the commit messages to determine the type of changes in the codebase. Following strict rules for commit messages, Semantic Release automatically determines the next semantic version number, generates a changelog and publishes the release.

By automating the versioning and publishing process, Semantic Release eliminates the immediate human error from these steps, ensures a more accurate and faster release cycle, and helps to focus on the work that adds value.
